Test inline scripts với nonce/hash - Demo cho đề tài Content Security Policy

Content-Security-Policy:
script-src 'self' 'nonce-{{NONCE}}' 'sha256-YOUR_HASH_HERE'

📌 Hướng dẫn test

  • Mở DevTools Console (F12)
  • Click các nút bên dưới để inject inline scripts vào trang
  • Quan sát console log và alerts để xem script nào chạy, nào bị chặn
  • CSP violations sẽ hiện trong Console và được ghi vào /report-log-realtime
PHẢI BLOCK Test 1: Inline script không nonce/hash
Script inline không có nonce hoặc hash → CSP phải chặn
const s1 = document.createElement('script');
s1.textContent = "console.log('❌ BLOCKED'); alert('Should NOT show!');";
document.documentElement.appendChild(s1);
⏳ Chưa test - Click nút bên dưới
PHẢI PASS Test 2: Inline script với nonce đúng
Script inline có nonce khớp với CSP header → Phải chạy thành công
const s2 = document.createElement('script');
s2.setAttribute('nonce', '{{NONCE}}'); // Lấy từ CSP header
s2.textContent = "console.log('✅ NONCE works!'); alert('✅ NONCE Works!');";
document.documentElement.appendChild(s2);
⏳ Chưa test - Click nút bên dưới
PHẢI PASS Test 3: Inline script với hash đúng
Script inline có hash SHA256 khớp với CSP → Phải chạy thành công
(Cần thêm hash vào CSP header trước - chạy node scripts/build-hash.js)
const s3 = document.createElement('script');
s3.textContent = "console.log('✅ HASH works!'); alert('✅ HASH Works!');";
document.documentElement.appendChild(s3);
⏳ Chưa test - Click nút bên dưới
PHẢI BLOCK Test 4: Inline script với nonce/hash sai
Script có nonce sai hoặc hash không khớp → CSP phải chặn
const s4 = document.createElement('script');
s4.setAttribute('nonce', 'WRONG_NONCE_123');
s4.textContent = "console.log('❌ Wrong nonce/hash'); alert('Should NOT show!');";
document.documentElement.appendChild(s4);
⏳ Chưa test - Click nút bên dưới

✅ Kết quả mong đợi

  • Test 1: Console báo lỗi CSP, alert KHÔNG hiện, status "❌ BLOCKED"
  • Test 2: Console log "✅ NONCE works!", alert hiện, status "✅ SUCCESS"
  • Test 3: Console log "✅ HASH works!", alert hiện (nếu đã config hash)
  • Test 4: Console báo lỗi CSP, alert KHÔNG hiện, status "❌ BLOCKED"
