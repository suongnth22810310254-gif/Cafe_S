🔒 Security Headers (Clickjacking + CSP)

Trang này được bảo vệ bởi 2 LỚP BẢO MẬT:

1️⃣ Clickjacking Protection:

  • X-Frame-Options: DENY
  • frame-ancestors 'none'

2️⃣ CSP Inline Script Protection:

  • script-src 'self' 'nonce-XXX'
  • Chặn inline JS độc hại, chỉ chạy script có nonce

Kết quả:

  • ✅ Không thể nhúng vào iframe (chặn clickjacking)
  • ✅ Không thể inject inline script (chặn XSS)
  • ✅ BẢO VỆ KÉP!

⚠️ Đây là nút nhạy cảm - clickjacking attack sẽ cố lừa user click vào đây!
