🎯 Attacker Page (Clickjacking Attempt)

Trang này cố gắng nhúng Protected Page vào iframe

⚔️ Attack Scenario

Mục tiêu: Kẻ tấn công muốn lừa user click vào nút "Delete Account" trên victim page.

Phương pháp: Nhúng victim page vào iframe, dùng CSS che iframe (opacity: 0), đặt nút fake trùng vị trí.

Kết quả mong đợi: Browser sẽ CHẶN iframe vì victim có header bảo vệ!

✅ Protection Works!

Iframe bên dưới sẽ KHÔNG load được vì Protected Page có header:

  • X-Frame-Options: DENY
  • Content-Security-Policy: frame-ancestors 'none'

Mở DevTools Console để xem lỗi browser báo.

❌ BLOCKED BY SECURITY HEADERS

🛡️ Browser Blocked the Iframe!

💡 What happened?

Browser nhận được response từ /clickjacking/protected với headers: 

X-Frame-Options: DENY
Content-Security-Policy: frame-ancestors 'none'

Browser từ chối render content trong iframe và hiện lỗi:

"Refused to display 'http://localhost:5000/clickjacking/protected' in a frame because it set 'X-Frame-Options' to 'deny'."

→ Tấn công THẤT BẠI! User được bảo vệ.
