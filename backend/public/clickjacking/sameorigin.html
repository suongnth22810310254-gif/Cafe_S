🔒 Security Configuration

Trang này có header:

  • X-Frame-Options: SAMEORIGIN
  • Content-Security-Policy: frame-ancestors 'self'

Kết quả:

  • ✅ CHO PHÉP nhúng từ http://localhost:5000
  • ❌ CHẶN nhúng từ domain khác (e.g., attacker.com)

💡 Use Cases

SAMEORIGIN hữu ích khi:

  • Bạn muốn nhúng trang vào dashboard/admin của chính bạn
  • Single Page Application (SPA) cần iframe internal pages
  • Multi-tab interface trong cùng 1 domain (ví dụ: Google Workspace)

Nếu muốn HOÀN TOÀN chặn iframe → dùng DENY / frame-ancestors 'none'
